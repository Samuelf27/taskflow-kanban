## 📌 Sobre o projeto

**TaskFlow** é um gerenciador de tarefas no estilo Kanban (como Trello), com três colunas — *A Fazer*, *Fazendo* e *Concluído*. Você cria cartões, arrasta entre colunas e tudo é salvo automaticamente no navegador. Pode ainda ser **instalado como aplicativo** (PWA) e funciona offline.

## ✨ Funcionalidades

- 🃏 Criar e remover cartões em cada coluna
- 🖱️ **Arrastar e soltar** entre colunas (HTML5 Drag & Drop)
- 💾 **Persistência automática** via `localStorage`
- 📲 **PWA instalável** com service worker (funciona offline)
- 🔢 Contador de tarefas por coluna
- 📱 Layout responsivo

## 🛠️ Tecnologias

HTML5 · CSS3 · JavaScript · Drag and Drop API · localStorage · Service Worker / PWA

## 🚀 Como rodar

```bash
git clone https://github.com/Samuelf27/taskflow-kanban.git
cd taskflow-kanban
# sirva os arquivos (o service worker exige http, não file://)
npx serve .
```

> 💡 Ao abrir pela versão hospedada, clique em **⬇️ Instalar app** para usá-lo como um aplicativo nativo.
